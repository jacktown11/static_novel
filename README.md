# html

pc 页面的主页和详情页公用顶部、底部，还有很多公用样式，因此写在一个页面里的，分别在两个 div 中。

# css

- `font.css`
  + 图标和自定义字体样式，结合 `font` 文件夹下的字体文件使用
  + 如果需要跨目录合并到其他 css 文件中，需要注意内部引用的字体文件路径是否保持正确
  + 自定义图标字体名字 `ICF`，使用的元素带用`ICF`类，图标字体类名前缀 `ic-`
- `common.css`
  + 全局样式重置、初始化、常用复用类名样式
- `main.css`、`detail.css`
  + 页面各自的样式
